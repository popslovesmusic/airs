Usage (current build)
---------------------
Configure (reusing downloaded GTest):
  cmake -S Simulation -B build -DBUILD_TESTS=ON -DBUILD_HARNESS=ON -DUSE_GTEST=ON \ 
        -DFETCHCONTENT_SOURCE_DIR_GOOGLETEST=D:/airs/build/_deps/googletest-src/googletest-1.14.0 \ 
        -DFETCHCONTENT_UPDATES_DISCONNECTED=ON

Build:
  cmake --build build --target harness_tests

Run tests (VS generator):
  cd build && ctest -C Debug -L harness

Artifacts/paths
---------------
- Binary: build/Debug/harness_tests.exe
- Metrics: metrics/<engine_family>/ (pre-created)
- Validation policy source: validation table.txt
- Harness sources: Simulation/tests/harness/

Notes
-----
- If using a different generator (e.g., Ninja), drop the -C Debug when running ctest.
- FetchContent download already completed; no further network needed unless _deps is cleaned.
